# Tower of Hanoi

Implementation of the Tower of Hanoi in C.

## Requirements

Basic:

- [gcc](gcc.gnu.org/)
- [make](https://www.gnu.org/software/make/)
- [cmake](cmake.org/)
- [doxygen](doxygen.nl/)
